# 2am — a quiet space for sleepless nights

> When your thoughts won't stop. Private, anonymous, nothing saved.

A calm, structured tool that walks you through 5 gentle questions when you can't sleep. It externalizes your thoughts, grounds you in the present moment, and reflects your own words back to you — quietly, without judgment.

**Nothing is ever saved. No account. No tracking. Closing the tab removes all traces.**

---

## Deploy in 5 minutes

### Option A — Vercel (recommended)

1. Push this folder to a GitHub repo
2. Go to [vercel.com](https://vercel.com) → "Add New Project"
3. Import your GitHub repo
4. Vercel auto-detects Vite — hit **Deploy**
5. You get a URL like `2am-tool.vercel.app`

### Option B — Netlify

1. Push this folder to a GitHub repo
2. Go to [netlify.com](https://netlify.com) → "Add new site" → "Import from Git"
3. Build command: `npm run build`
4. Publish directory: `dist`
5. Hit **Deploy**

### Option C — Manual (any static host)

```bash
npm install
npm run build
# Upload the /dist folder to any static host
```

## Stack

- React 18
- Vite 5
- Zero dependencies beyond React
- No backend, no database, no analytics
- Fonts: EB Garamond + DM Mono (Google Fonts)

## Philosophy

Most wellness apps want your data, your subscription, your engagement. This one wants nothing from you. It exists for the 2am moment when you need somewhere to put your thoughts — and then it lets them go.

Built with care. Use freely.
